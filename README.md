Link Image Viewer
===
Description
-------------------------
Link Image Viewer is an application that shows pictures from links. It consists of 2 applications: A - main, and B - dependent from A. Application A has two tabs: test (addition new links) and history (to look at the added links). Application B shows image from link and download it into phone storage.

Features
-------------------------
* Show pictures from links.
* Save history of added links.
* Sorting history by addition time and addition status.
* Download images from links to SD card (or to internal storage, if SD card is absent).
* Availability of a database in which all the necessary information is stored
